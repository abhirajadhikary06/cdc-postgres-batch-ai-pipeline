# cdc-postgres-batch-ai-pipeline
A modern CDC Postgres AI batch pipeline built with PostgreSQL, CDC ingestion tools, a medallion architecture data layer using MinIO, Iceberg, and Polars, with metadata management, data warehousing, AI analytics, and Docker for containerized deployment.
